</span><a href="http://blog.freenode.net/2014/11/atheme-7-2-and-freenode/" title="23:18" rel="bookmark"><time class="entry-date" datetime="2014-11-08T23:18:51+00:00">2014-11-08</time></a><span rel="author">tomaw</a></span></span>  </div><!-- .entry-meta -->    </header><!-- .entry-header -->   <div class="entry-content">   <p>Hello!</p> <p>We&#8217;ve begun some testing on Atheme&#8217;s latest release, 7.2, and we&#8217;d like to invite interested users to help with that.</p> <p>Not all changes the Atheme project has included in their new release will be included in our Atheme upgrade, so here&#8217;s the bulk of the changes that will actually effect our network:</p> <ul> <li>/msg NickServ DROP will require confirmations from the user similar<br /> to the ChanServ variant. This is to prevent people DROPping when they<br /> should be GHOSTing or similar.</li> <li> We&#8217;ve loaded two exttargets: <ul> <li>$registered to grant flags to all people who are identified to<br /> NickServ</li> <li>$chanacs to grant flags to people who have flags in another<br /> channel. Please read /msg ChanServ HELP FLAGS for details on how they work.</li> </ul> </li> <li>The SASL mechanism DH-BLOWFISH has been removed. People using it<br /> can connect via SSL and use PLAIN or upgrade to ECDSA-NIST256P-CHALLENGE.<br /> Details of how to do so are <a href="https://github.com/atheme/ecdsatool">here</a> and <a href="https://freenode.net/sasl/">our SASL page</a> will be updated with the relevant documentation soonish.</li> </ul> <p>You should be able to connect to testnet at testnet.freenode.net Port 9002 for cleartext, and 9003 for SSL. Bear in mind, the database is a couple weeks old, so changes you&#8217;ve recently made on the production network may not be mirrored on the testnet network. Various amounts of staff should be idling in #freenode on testnet at all times, please feel free to poke us with any questions.</p> <p>Thanks!</p> <p>&nbsp;</p>    </div><!-- .entry-content -->   <footer class="entry-meta">   This entry was posted in <a href="http://blog.freenode.net/category/uncategorized/" rel="category tag">Uncategorized</a> by <a href="http://blog.freenode.net/author/tomaw/">tomaw</a>.
