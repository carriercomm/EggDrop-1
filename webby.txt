/></div>   <div id="content">         <h2>Welcome to Zandronum!</h2>         <h3>Leading the way in newschool multiplayer Doom online.</h3>         <p>Zandronum is a multiplayer oriented port, based off <a href="http://skulltag.com">Skulltag</a>, for Doom and Doom II by <a href="http://www.idsoftware.com/">id Software</a>.</p>         <p>Zandronum brings classic Doom into the 21st century, maintaining the essence of what has made Doom great for so many years and, at the same time, adding new features to modernize it, creating a fresh, fun new experience.</p>         <p>Here's why you should kill your time with Zandronum, versus vanilla Doom:</p>         <ul>                 <li><b>Excellent online play</b>                         <ul>                                 <li>Client/server architecture: play and switch between games with ease</li>                                 <li>Supports up to 64 players                                         <ul>                                                 <li>Wide array of gameplay modes</li>                                                 <li>Competitive: Team DM, Duel, CTF, Possession, LMS, Terminator, Skulltag</li>                                                 <li>Cooperative: Survival, Invasion (or both!)</li>                                                 <li>Modifiers, like Instagib and Buckshot, can be used to spice up any game.</li>                                         </ul>                                 </li>                         </ul>                 </li>                 <br />                 <li><b>Modern and beautiful</b>                         <ul>                                 <li>Support for large number of <a href="http://zdoom.org">ZDoom</a> and <a href="http://www.osnanet.de/c.oelckers/gzdoom/index.html">GZDoom</a> mods</li>                                 <li>Choose between GZDoom's OpenGL renderer, or Doom's Software renderer</li>                                 <li>Useful features like a console, Quake-style key bindings, freelooking, and jumping</li>                                 <li>Support for many Doom engine based games including Heretic, Hexen, and Strife</li>                                 <li>Supports modern versions of Windows, Mac OS X, and Linux.</li>                         </ul>                 </li>                 <br />                 <li><b>Compatible with Skulltag mods</b>                         <ul>                                 <li>Supports announcers and skins for online play</li>                                 <li>Runes, new weapons, and new monsters for modders</li>                                 <li>Bots, and support for custom bots</li>                         </ul>                 </li>                 <br />                 <li><b>Open Source</b>                         <ul>                                 <li>Doesn't run on your flavor of operating system?     <br />     The source code
