style="float:left;margin-top:0px;margin-right:4px;margin-left:4px;"> <style type="text/css"> .adslot_1 { width: 234px; height: 90px; } @media (min-width:1000px) { .adslot_1 { width: 300px !important; Test --> <ins class="adsbygoogle adslot_1"     style="display:inline-block;"          data-ad-client="ca-pub-5406364303442190"      data-ad-slot="6144069130"></ins> <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script> <script>(adsbygoogle = window.adsbygoogle || []).push({}); </script> </div>   <p>Up until now it was quite difficult to get your hands on a proper installation disc or USB flash drive to install Windows 8.1 on a PC.</p> <p>While you can buy a Windows 8.1 DVD, you don't get media if you upgrade Windows 8 to Windows 8.1 for example. If you then decide to format the system and start anew or move everything to a new PC, you may have noticed that you need to install Windows 8 first to update it to Windows 8.1.</p> <p>Downloading media can also be useful if you cannot install Windows 8.1 from Windows Store for whatever reason or don't want to perform that operation for multiple systems that you want to update.</p> <p>Workarounds have been available for some time to <a href="http://www.ghacks.net/2013/10/19/download-windows-8-1-iso-image/">create Windows 8.1 installation media</a> to avoid going through this lengthy installation process but they too required you to go through a lengthy process.</p> <p>Microsoft is now making available a <a href="http://windows.microsoft.com/en-us/windows-8/create-reset-refresh-media" title="" class="ext-link" target="_blank">Media Creation Tool</a> which you can use to create Windows 8.1 installation media right away.</p> <p>After you have downloaded the program from the official website and started it, you are asked to select the language, edition and architecture.</p> <p>All languages seem to be supported. As far as editions are concerned, you get Windows 8.1 and Windows 8.1 Pro, the N versions of both operating systems, and a single language Windows 8.1 version.</p> <p><a href="http://cdn.ghacks.net/wp-content/uploads/2014/11/windows-installation-media-creation-tool.jpg"><img class="alignnone size-full wp-image-107046" src="http://cdn.ghacks.net/wp-content/uploads/2014/11/windows-installation-media-creation-tool.jpg" alt="windows installation media creation tool" width="716" height="561" /></a></p> <p>N editions of Windows 8.1 don't include any media related technologies such as windows Media Player or Skype.</p> <p>A click on next displays the available save options. You can save the installation files to a USB drive right away provided that it offers at least 4 Gigabyte of space or create an ISO file instead which you need to burn to DVD then.</p> <p>You are then asked to select the USB Flash Drive you want to copy the Windows 8.1 installations files to or a save location for the ISO file.</p> <p><a href="http://cdn.ghacks.net/wp-content/uploads/2014/11/windows-8.1-iso-usb.jpg"><img class="alignnone size-full wp-image-107048" src="http://cdn.ghacks.net/wp-content/uploads/2014/11/windows-8.1-iso-usb.jpg" alt="windows 8.1 iso usb" width="716" height="561" /></a></p> <p>The program displays the download progress in its interface then and estimates the remaining time until the operation completes.</p> <p><a href="http://cdn.ghacks.net/wp-content/uploads/2014/11/download-windows-8.1-media.jpg"><img class="alignnone size-full wp-image-107047" src="http://cdn.ghacks.net/wp-content/uploads/2014/11/download-windows-8.1-media.jpg" alt="download windows 8.1 media" width="716" height="561" /></a></p> <p>The USB Flash Drive can be used right away while it is necessary to burn the ISO image to a DVD first.</p> <p>It took a while for Microsoft to provide Windows customers with an option to download a Windows 8.1 ISO image from an official source.</p> <p>The release of the Media Creation Tool provides all users with the option to create ISO images of Windows 8.1 or copy the installation files to a Flash Drive instead.</p> <div id="snippet-box" style="background:#F5F5F5; color:#333333; border:1px solid #ACACAC;"><div class="snippet-title" style="background:#E4E4E4; color:#333333; border-bottom:1px solid
